# 📘 Life Caddie Estimator

Main repo used for the creation and testing of the Life Caddie estimator app.

## 📝 Overview

The app is to implement modern image recognition techniques to make **effective organizing** estimations for rooms and spaces.

## 🏗️ Tech Stack

* Python [3.12.3]
